# Kardex de un Alumno

## 📘 Descripción del Proyecto
Este proyecto es una aplicación en Java que calcula el promedio de cinco calificaciones de un alumno, determina su calificación final basada en una tabla de rangos y muestra los resultados en un formato específico.

## 🛠️ Instrucciones para Compilar y Ejecutar

1. Abre una terminal y navega a la carpeta donde está el archivo `.java`.
2. Compila el archivo con el siguiente comando:

   ```bash
   javac Kardexdeunalumno.java
   ```
